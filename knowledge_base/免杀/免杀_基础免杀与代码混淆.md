# 基础免杀与代码混淆技巧

## 常见静态免杀方法

### 1. 字符串混淆

**原始 C# shellcode 加载器**：
```csharp
byte[] buf = new byte[523] { 0xfc,0x48,0x83,... };
```



**混淆后**：

csharp

```
string shellcodeHex = "fc4883e4f0e8c...";
byte[] buf = new byte[shellcodeHex.Length / 2];
for (int i = 0; i < shellcodeHex.Length; i+=2)
    buf[i/2] = Convert.ToByte(shellcodeHex.Substring(i,2), 16);
```



### 2. 编码与加密

- **Base64** 编码后再解码执行
- **XOR** 单字节异或：`shellcode[i] ^= 0xAA`
- **AES** 解密 (需要提供密钥)

### 3. 入口点伪装

- 使用合法图标（如 PDF 图标）
- 添加数字签名（伪造或窃取）
- 增加无用的循环运算延缓沙箱检测

## 内存执行技术

### Process Hollowing (进程镂空)

1. 创建挂起的合法进程 (如 `svchost.exe`)
2. 卸载其原始代码
3. 写入 shellcode 并恢复线程

### 使用 DInvoke

代替 `VirtualAllocEx` 等敏感 API，通过动态调用未文档化函数。

## 注意事项

- 免杀不是一劳永逸，需结合 **分离加载**（从远程服务器拉取 shellcode）
- 测试环境：Windows Defender + 一款主流杀软（卡巴/火绒/360）
- 优先使用 **回调函数执行**（如 `EnumWindows`、`CreateThreadPoolWait`）避免直接调用 `CreateThread`
