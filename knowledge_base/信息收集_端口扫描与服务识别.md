# 端口扫描与服务识别技巧

## 常用扫描工具

### Nmap 基本用法
```bash
# 快速 TCP 全端口扫描
nmap -sS -p- -T4 --min-rate=1000 192.168.1.1 -oA fast_scan

# 服务版本和操作系统探测
nmap -sV -sC -O -p 22,80,443,445,3389 192.168.1.1

# UDP 关键端口扫描
nmap -sU --top-ports 20 192.168.1.1
```



### 规避 IDS/防火墙

bash

```
# 诱饵扫描
nmap -D RND:10,ME 192.168.1.1

# 分段扫描（慢速）
nmap -T2 --scan-delay 5s -p 1-1000 192.168.1.1

# 使用非标准协议（如 FTP bounce）
nmap -b ftp:anonymous:password@192.168.1.2 192.168.1.1
```



## 服务指纹深度识别

| 服务 | 常用端口    | 识别技巧                 |
| :--- | :---------- | :----------------------- |
| Web  | 80,443,8080 | 使用 whatweb, wappalyzer |
| SMB  | 445         | `smbclient -L //IP`      |
| SSH  | 22          | 版本直接连接即可获取     |
| RDP  | 3389        | `rdp-sec-check.pl` 脚本  |

## 被动信息收集

- **Shodan**：`ssl:"Self signed"` 搜索自签名证书设备
- **Censys**：查询特定组织的暴露资产
- **DNS 历史记录**：SecurityTrails 查找子域名

**小贴士**：内网扫描时优先使用 ARP 扫描 (`nmap -sn -PR`)，速度远超 ICMP。
